# NPG Kanban Task Board

Full-stack Kanban board: **React + TypeScript** (Vite), **ASP.NET Core 8** API, **Supabase** (database, anonymous auth, RLS).

## Quick start

See **[SETUP.md](./SETUP.md)** for Supabase SQL, environment variables, local run, and optional hosting notes.

## Repository layout

- `NPGKanbanStyleTaskBoard.Server` — Web API  
- `npgkanbanstyletaskboard.client` — React SPA  
- `Dockerfile` — API image for container hosts (e.g. Render)
